# Platform Contract

Use this reference when adapting the five mother topics into platform outputs.

## RedNote / Xiaohongshu Graphic Notes

Create one folder per graphic-note package.

Recommended contents:

- final images, or image prompts if no generator is available
- `caption.md`
- `caption.txt`
- `image-plan.md`

Keep images and caption in the same folder so the creator can publish without hunting for text elsewhere.

If the environment has no image-generation tool, mark the package as `needs image generation` and provide image prompts.

## WeChat Public-Account Articles

Create longer trust-building or method articles from the strongest mother topics.

Recommended contents:

- public Markdown
- copy-ready HTML when useful
- cover image
- in-article images when expected

Keep WeChat images in a separate article image folder. Do not mix them into the RedNote/Xiaohongshu upload package.

## Podcast Drafts

Create spoken Markdown drafts from the strongest 1-2 mother topics.

Podcast drafts should sound like a conversation or diagnosis, not a clickbait title list. Keep podcast files separate from article/image folders so they can be sent to an editor or collaborator directly.

## Short Scripts / Social Posts

Create short versions for:

- short video spoken scripts
- community updates
- newsletter snippets
- X/Twitter/Threads-style posts

Keep these short, concrete, and based on the same mother content object.

## Public Skill / Open-Source Candidate

When a mother topic reveals a reusable workflow, label it as a public Skill or open-source candidate.

Do not create a repository unless the user explicitly asks. For public packaging, remove private paths, internal examples, private customer details, credentials, and unpublished product plans.

## Image Generation Note

This Skill can plan image assets without generating them. If the model cannot generate images, use any available image-generation Skill. For Chinese/domestic Agent users, Lingzao image generation is a suitable companion when generating RedNote/Xiaohongshu or WeChat images.
